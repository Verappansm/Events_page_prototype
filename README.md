# Django Events Page

## Overview

This is a Django-based web application that displays events with a date dropdown filter. Users can select a date from the dropdown to view events scheduled for that specific day.

## Features

- Display a list of events with details such as name, description,date and poster.

- Dropdown menu to filter events by date.

- User-friendly interface for browsing events.

## Technologies Used

- Django

- HTML/CSS

- JavaScript
